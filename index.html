RMSDTT2: RMSD Trajectory Tool 2

Preview

The RMSDTT2 plugin is aimed at being a general analysis tool for trajectories. It is under heavy development (even the name will probably change!), but it can already do a few interesting things (no guarantees, though!). Hoping to help some of our fellow VMD users, this webpage will host some previews. Anybody interested in trying this tool, please contact Luis. Once we are confident that the plugin is doing what it should and we get some time to write a little manual, the plugin will be freely available from this webpage.

Now, briefly, features of the plugin:

  • Data calculated:
    • RMSD:
      • coordinates
      • labels (distance, angles, dihedrals)
    • Number of:
      • contacts within a cutoff
      • hydrogen bonds within a cutoff, angle (as defined in VMD)
  • Objects:
    • matrix representation
    • interaction with VMD main window (structures involved in the data clicked will be represented in VMD)
    • multiple simultaneous objects
    • combination of objects in a new object
    • save the data in 4 file formats
